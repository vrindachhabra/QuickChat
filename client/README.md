Full Stack Real-Time Chat Application (MERN + Socket.IO)

A full-stack real-time chat application built using the *MERN stack* (MongoDB, Express, React, Node.js) with *Socket.IO* for instant bi-directional communication.  
Users can send and receive messages instantly without reloading the page.

Tech Stack-

Frontend:
- React.js
- Socket.IO Client
- CSS / Tailwind
- Axios
Backend:
- Node.js
- Express.js
- Socket.IO
Database:
- MongoDB
